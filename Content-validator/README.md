Content Validator
===============

Instructions:
1. Run the validator (double-click Run_Content_Validator.bat)
2. Review the validation report in your Documents\Content_Validation folder

The report will check for:
- HTML structure issues
- CSS problems
- Accessibility concerns
- Content structure
- Interactive elements

Reports are saved as CSV files with timestamps in your Downloads folder.

For questions or to add new rules, contact @bukvic.
